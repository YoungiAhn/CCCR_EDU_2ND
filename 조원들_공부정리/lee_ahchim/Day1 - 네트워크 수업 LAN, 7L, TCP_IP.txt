시스템 : 능동적으로 정보를 처리할 수 있는 기기. 네트워크에 연결되어 정보를 공유할 수 있음. 단말기(Terminal Device), 즉 End-Point도 시스템과 유사어라고 볼 수 있다.
네트워크 인터페이스 (컨트롤러) : NIC라고 부르며, 시스템을 네트워크에 연결하여 통신하기 위해 사용되는 하드웨어 장치, 즉 랜카드를 의미
네트워크 전송 매체 : UDP 케이블 혹은 공기/진공 등을 일컫는 말.
(통신) 프로토콜 : 시스템과 원거리 통신 장비 사이에서 데이터를 주고받을 수 있는 규칙 혹은 체계.






네트워크 : 전송 매체(UDP 케이블 or 공기/진공)로 서로 연결된 두 대 이상의 시스템의 모음
비트 : 컴퓨터의 정보를 나타내는 최소 단위
바이트 : 컴퓨터가 조작하는 정보의 최소 처리 단위. 비트의 모음.
LAN : Local Area Network. 거리가 가까운 시스템끼리 같은 네트워크 대역대로 연결한 것.
WAN : Wide Area Network. 두 개 이상의 LAN을 연결해 각 LAN에 포함되어 있는 A 시스템과 B 시스템 간 서로 통신이 가능하게 하는 네트워크.
ISP : Internet Service Provider. 인터넷 서비스 공급자를 의미, 인터넷 회사라고 많이들 일컬음. 집이나 사업장에서 외부와 통신할 수 있도록 유료로 인터넷을 제공해 주며 IP를 할당해주는 공급자.

Server : 특정 서비스나 정보를 고객이 사용할 수 있도록 제공하는 단말기 혹은 시스템.. Web Server FTP Server Mail Server 등등
Client : 서버가 제공하는 서비스를 사용하는 고객 쪽 단말기 혹은 시스템
